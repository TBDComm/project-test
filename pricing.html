자주 묻는 질문

해지하면 어떻게 되나요? +
언제든 해지할 수 있습니다. 해지해도 결제한 달 말까지는 모든 기능이 유지됩니다. 다음 달부터 무료 플랜으로 전환됩니다.
무료 플랜 5회는 어떻게 계산되나요? +
Spotlight Compare를 분석할 때마다 1회가 차감됩니다. 매월 1일에 5회로 초기화됩니다.
어떤 결제 수단을 지원하나요? +
신용카드, 체크카드, 간편결제(카카오페이, 토스페이 등)를 지원합니다. 토스페이먼츠를 통해 안전하게 결제됩니다.
데이터는 얼마나 최신 정보인가요? +
YouTube Data API v3를 사용해 실시간으로 데이터를 가져옵니다. Spotlight Compare는 최근 7일 내 인기 영상, Timing Report는 최근 30일 기준입니다.
환불이 가능한가요? +
결제일로부터 7일 이내, 서비스를 5회 이하 사용한 경우 전액 환불이 가능합니다. 환불 문의는 support@momento.kr로 연락해주세요.
